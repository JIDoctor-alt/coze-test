### 1. SM4Util.java - 国密SM4加密工具

> 🎯 中国国家密码管理局发布的加密算法

#### 这是啥？（小白版）
就像古代的**虎符密码**，只有持有正确的一半才能解开：
- 加密：把信息变成乱码
- 解密：把乱码还原成信息

#### 核心代码解析

```java
public class SM4Util {
    
    // 私钥（必须是16字节）
    private static final String testKey = "1234567812345678";
    
    static {
        // 添加BouncyCastle安全提供者（SM4算法需要）
        Security.addProvider(new BouncyCastleProvider());
    }
    
    // 加密
    public static String sm4Encrypt(String plainString) {
        // 1. 指定算法：SM4
        String algorithm = "SM4";
        
        // 2. 创建密钥
        SecretKeySpec secretKeySpec = new SecretKeySpec(
            testKey.getBytes(StandardCharsets.UTF_8), algorithm);
        
        // 3. 获取Cipher（SM4默认ECB+PKCS5Padding）
        Cipher cipher = Cipher.getInstance(algorithm + "/ECB/PKCS5Padding");
        
        // 4. 初始化为加密模式
        cipher.init(Cipher.ENCRYPT_MODE, secretKeySpec);
        
        // 5. 执行加密
        byte[] cipherBytes = cipher.doFinal(
            plainString.getBytes(StandardCharsets.UTF_8));
        
        // 6. Base64编码输出
        return Base64.getEncoder().encodeToString(cipherBytes);
    }
    
    // 解密
    public static String sm4Decrypt(String cipherString) {
        // 类似解密流程...
    }
}
```

#### 使用场景
```java
// 用户密码加密
String encryptedPassword = SM4Util.sm4Encrypt("PICChealth@2020");

// 密码验证
String inputPassword = SM4Util.sm4Encrypt("userInput");
boolean matches = encryptedPassword.equals(inputPassword);
```

### 2. UserUtils.java - 用户上下文工具 ⭐核心

> 🎯 获取当前登录用户的信息

#### 这是啥？（小白版）
就像公司的**员工胸卡扫描器**，一扫描就知道你是谁、在哪个部门。

#### 核心代码解析

```java
public class UserUtils {
    
    // 使用ThreadLocal存储用户信息（线程隔离）
    public static ThreadLocal<User> threadLocal = new ThreadLocal<>();
    
    // 获取当前登录用户
    public static User getUser() {
        User localUser = threadLocal.get();
        if (localUser == null) {
            // 如果没有登录，返回默认用户（避免空指针）
            localUser = new User();
            localUser.setUserId("000000");
            localUser.setUserName("人保健康");
            localUser.setUserAccount("123456");
            localUser.setOrgId("866103");
            localUser.setOrgName("中科软");
        }
        return localUser;
    }
    
    // 设置当前用户
    public static void setUser(User user) {
        threadLocal.set(user);
    }
    
    // 清除用户信息
    public static void remove() {
        threadLocal.remove();
    }
}
```

#### 使用场景
```java
// 在Service中使用
public void createUser() {
    String currentUser = UserUtils.getUser().getUserAccount();
    // 设置创建人
    user.setCreator(currentUser);
}

// 在Interceptor中设置
public class TokenInterceptor {
    public void preHandle() {
        User user = getUserFromToken(token);
        UserUtils.setUser(user);  // 存入ThreadLocal
    }
    
    public void afterCompletion() {
        UserUtils.remove();  // 请求结束后清理
    }
}
```

### 4. EncryptionUtil.java - 通用加密工具

> 🎯 MD5、SHA等加密算法

#### 核心代码解析

```java
public class EncryptionUtil {
    
    // MD5加密（32位十六进制）
    public static String encodeMD5(String inStr) {
        MessageDigest md5 = MessageDigest.getInstance("MD5");
        byte[] md5Bytes = md5.digest(inStr.getBytes("UTF-8"));
        
        StringBuffer hexValue = new StringBuffer();
        for (int i = 0; i < md5Bytes.length; i++) {
            int val = ((int) md5Bytes[i]) & 0xff;
            if (val < 16) {
                hexValue.append("0");
            }
            hexValue.append(Integer.toHexString(val));
        }
        return hexValue.toString();
    }
    
    // SHA-256加密（64位十六进制）
    public static String getPassword(String pwd) {
        byte[] hashPassword = getMessageDigest("SHA-256", pwd.getBytes());
        return binaryToHexString(hashPassword);
    }
}
```

## ThreadLocal原理

```
┌─────────────────────────────────────────────────────────┐
│              ThreadLocal工作原理                          │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  请求1 (Thread-1)                                        │
│       ↓                                                  │
│  UserUtils.setUser(user1)                               │
│       ↓                                                  │
│  threadLocal.set(user1)  →  Thread-1的map中存: user1     │
│                                                          │
│  请求2 (Thread-2)                                        │
│       ↓                                                  │
│  UserUtils.setUser(user2)                               │
│       ↓                                                  │
│  threadLocal.set(user2)  →  Thread-2的map中存: user2     │
│                                                          │
│  结果：每个线程访问自己的用户，互不干扰                     │
│                                                          │
└─────────────────────────────────────────────────────────┘
```
