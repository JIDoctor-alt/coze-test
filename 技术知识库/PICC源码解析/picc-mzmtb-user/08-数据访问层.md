# 08 - 数据访问层 (DAO)

> 🎯 一句话说明：MyBatis数据库操作层，负责与数据库的交互

---

## 模块概述

这是系统的**数据库操作层**，类似于医院的**病历档案室管理员**：
- 查找档案 → SELECT查询
- 新建档案 → INSERT插入
- 修改档案 → UPDATE更新
- 销毁档案 → DELETE删除

**生活比喻**：就像图书馆的**管理员系统**
- 查找书籍 → 查询数据库
- 借书/还书 → 增删改操作
- 图书分类 → 数据组织

## MyBatis Mapper XML示例

### PrivilegeUserInfoMapper.xml

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE mapper PUBLIC "-//mybatis.org//DTD Mapper 3.0//EN" 
    "http://mybatis.org/dtd/mybatis-3-mapper.dtd">

<mapper namespace="com.picchealth.module.system.dao.PrivilegeUserInfoDao">

    <!-- 根据账号查询用户 -->
    <select id="selectByAccount" resultMap="BaseResultMap">
        SELECT * FROM privilege_user_info 
        WHERE account = #{account} 
        AND delete_at IS NULL
    </select>
    
    <!-- 条件查询用户 -->
    <select id="queryUserInfo" resultMap="BaseResultMap">
        SELECT * FROM privilege_user_info 
        WHERE delete_at IS NULL
        <if test="email != null and email != ''">
            AND email = #{email}
        </if>
        <if test="tel != null and tel != ''">
            AND tel = #{tel}
        </if>
        <if test="account != null and account != ''">
            AND account = #{account}
        </if>
    </select>
    
    <!-- 根据机构查询用户 -->
    <select id="selectByOrgId" resultMap="BaseResultMap">
        SELECT * FROM privilege_user_info 
        WHERE org_id = #{orgId} 
        AND delete_at IS NULL
    </select>

</mapper>
```

## 常用SQL操作

### 1. 插入操作
```xml
<!-- 单条插入 -->
<insert id="insert" parameterType="PrivilegeUserInfo">
    INSERT INTO privilege_user_info (
        id, account, name, password, tel, email, org_id
    ) VALUES (
        #{id}, #{account}, #{name}, #{password}, 
        #{tel}, #{email}, #{orgId}
    )
</insert>

<!-- 选择性插入（只插入非空字段） -->
<insert id="insertSelective" parameterType="PrivilegeUserInfo">
    INSERT INTO privilege_user_info
    <trim prefix="(" suffix=")" suffixOverrides=",">
        <if test="id != null">id,</if>
        <if test="account != null">account,</if>
        <!-- 更多字段... -->
    </trim>
    <trim prefix="VALUES (" suffix=")" suffixOverrides=",">
        <if test="id != null">#{id},</if>
        <if test="account != null">#{account},</if>
        <!-- 更多字段... -->
    </trim>
</insert>
```

### 2. 更新操作
```xml
<!-- 更新用户 -->
<update id="updateByPrimaryKey" parameterType="PrivilegeUserInfo">
    UPDATE privilege_user_info
    SET 
        name = #{name},
        tel = #{tel},
        modifier = #{modifier},
        modify_time = NOW()
    WHERE id = #{id}
</update>

<!-- 软删除 -->
<update id="softDelete">
    UPDATE privilege_user_info
    SET delete_at = NOW(),
        modifier = #{modifier}
    WHERE id = #{id}
</update>
```

### 3. 查询操作
```xml
<!-- 主键查询 -->
<select id="selectByPrimaryKey" resultMap="BaseResultMap">
    SELECT * FROM privilege_user_info 
    WHERE id = #{id}
</select>

<!-- 条件查询 -->
<select id="select" parameterType="PrivilegeUserInfo" resultMap="BaseResultMap">
    SELECT * FROM privilege_user_info 
    WHERE delete_at IS NULL
    <if test="account != null">
        AND account = #{account}
    </if>
    <if test="name != null">
        AND name LIKE CONCAT('%', #{name}, '%')
    </if>
</select>

<!-- 分页查询 -->
<select id="selectPage" resultMap="BaseResultMap">
    SELECT * FROM privilege_user_info 
    WHERE delete_at IS NULL
    LIMIT #{offset}, #{limit}
</select>
```

## 软删除实现

### 什么是软删除？
```sql
-- 硬删除（物理删除，数据没了）
DELETE FROM table WHERE id = '123';

-- 软删除（逻辑删除，设置标记，数据还在）
UPDATE table SET delete_at = NOW() WHERE id = '123';
```

### 软删除的好处
1. **数据可恢复** - 误删可以恢复
2. **保留历史** - 便于审计追踪
3. **关联保留** - 不影响历史数据关联

### 本项目软删除字段
```sql
-- 所有主要业务表都有delete_at字段
ALTER TABLE privilege_user_info ADD COLUMN delete_at DATETIME;
ALTER TABLE privilege_org_info ADD COLUMN delete_at DATETIME;
ALTER TABLE privilege_role_info ADD COLUMN delete_at DATETIME;
ALTER TABLE privilege_menu_info ADD COLUMN delete_at DATETIME;
```

## BaseDao通用接口

```java
public interface BaseDao<T> {
    // 插入
    int insert(T entity);
    int insertSelective(T entity);
    
    // 更新
    int updateByPrimaryKey(T entity);
    int updateByPrimaryKeySelective(T entity);
    
    // 删除
    int deleteByPrimaryKey(String id);
    
    // 查询
    T selectByPrimaryKey(String id);
    List<T> select(T entity);
    T selectOne(T entity);
}
```
