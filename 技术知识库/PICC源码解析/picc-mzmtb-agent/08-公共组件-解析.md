# 公共组件解析 - components/

> 🎯 一句话说明：公共组件是"可复用的积木块"，在不同页面中可以重复使用

---

## 这是啥？（小白版）

想象 **公共组件是乐高玩具的"标准零件"**：
- 搜索框 = 搜索卡片
- 弹窗 = 模态框
- 表格 = 数据展示
- 同一个零件可以在多个作品中重复使用

## 1. searchCard 搜索卡片组件

### 位置
`components/searchCard/index.vue`

### 功能
通用的查询条件输入组件，包含表单验证和重置功能。

### 使用方式
```html
<searchCard
    :formObj="formObj"           <!-- 查询条件对象 -->
    :bjUnitList="unitList"       <!-- 宝鸡单位列表 -->
    :bjPersontypeList="persontypeList"  <!-- 人员类型列表 -->
    @search="search"            <!-- 查询事件 -->
    @resetFields="resetFields"   <!-- 重置事件 -->
    :colSpan="6"                 <!-- 列宽 -->
    :gutter="30"                 <!-- 间距 -->
/>
```

### 核心代码
```javascript
export default {
    props: {
        formObj: Object,           // 表单数据对象
        bjUnitList: Array,         // 宝鸡定点单位
        bjPersontypeList: Array,   // 人员类型
        colSpan: { type: Number, default: 6 },
        gutter: { type: Number, default: 20 }
    },
    data() {
        return {
            form: this.$form.createForm(this)  // 创建表单实例
        }
    },
    methods: {
        // 查询按钮点击
        handleSearch() {
            this.form.validateFields((err, values) => {
                if (!err) {
                    this.$emit('search', values);  // 触发查询事件
                }
            });
        },
        // 重置按钮点击
        handleReset() {
            this.form.resetFields();
            this.$emit('resetFields');
        }
    }
}
```

## 5. tableMixins 表格混入

### 位置
`utils/tableMixins.js`

### 功能
封装了表格的通用逻辑，如分页、搜索等。

### 使用方式
```javascript
import tableMixins from "@/utils/tableMixins";

export default {
    mixins: [tableMixins],
    data() {
        return {
            pagination: {
                current: 1,
                pageSize: 10,
                total: 0
            }
        }
    },
    methods: {
        // 切换每页条数
        onPageSizeChange(current, pageSize) {
            this.pagination.pageSize = pageSize;
            this.pagination.current = 1;
            this.initPage();
        },
        // 切换页码
        onPageChange(page, pageSize) {
            this.pagination.current = page;
            this.initPage();
        }
    }
}
```

## 知识点

### 🔹 mixins 混入
```javascript
import tableMixins from "@/utils/tableMixins";

export default {
    mixins: [tableMixins]  // 混入通用逻辑
}
```
- 混入的 data、methods、生命周期钩子会合并
- 类似"继承"，但更灵活

### 🔹 $emit 事件
```javascript
// 子组件中触发
this.$emit('search', values);

// 父组件中监听
@search="search"  <!-- 或者 -->
this.$on('search', this.search);
```

### 🔹 props 传值
```javascript
// 父组件传值
<child :message="msg" />

// 子组件接收
props: {
    message: String,
    required: true
}
```
