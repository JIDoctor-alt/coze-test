|------|------|------|
| id | 主键ID | 1001 |
| name | 患者姓名 | 张三 |
| idcard | 身份证号 | 610302... |
| mobile | 手机号 | 1380000... |
| icdname | 疾病名称 | 高血压 |

| expert_status | 专家状态 | 0 |

|------|------|

| managername | 负责人 |

## 🔧 常用操作速查

### 1. 新增一个API接口
```javascript
// 1. 在 api/ 下新建文件 apiXXX.js
// 2. 编写接口方法
export default {
    getList: (params) => {
        return axios.post('/api/xxx/list', params);
    }
}
```

### 2. 新增一个页面路由
```javascript
// router/childrenRoutes.js
{
    path: "/newPage",
    name: "newPage",
    meta: { ptitle: "父菜单", title: "页面标题" },
    component: () => import("@/pages/xxx/newPage")
}
```

### 3. 新增一个API调用
```javascript
// 在组件中
import api from "@/api/apiXXX";

api.getList(params).then(res => {
```
