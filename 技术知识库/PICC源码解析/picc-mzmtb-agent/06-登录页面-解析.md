# 登录页面解析 - components/loginMb/index.vue

> 🎯 一句话说明：登录页是系统的"大门"，负责验证用户身份并发放"通行证"

---

## 这是啥？（小白版）

想象 **登录页是医院的"挂号窗口"**：
- 挂号窗口 = 登录页
- 身份证/医保卡 = 用户名密码
- 挂号单 = token（通行证）
- 挂号员 = 后端接口

---

### template 部分

### script 部分

```javascript
export default {
    data() {
        return {
            showCode: true,
            loginLoading: false,     // 登录按钮loading
            codeLoading: false,       // 验证码按钮loading
            codeButtonText: "发送验证码",  // 验证码按钮文字
            mobile: "",              // 自动填充的手机号
            resetModal: false,       // 重置密码弹窗
            resetLoading: false,
            captchaImgMB: ""         // 图形验证码图片
        }
    },
    methods: {
        // 1. 登录提交
        handleSubmit(e) {
            e.preventDefault();
            this.form.validateFields((err, values) => {
                if (!err) {
                    this.loginLoading = true;
                    api.login(values).then(res => {
                        if (res.status === 0) {
                            // 登录成功
                            const { token, userId, userName } = res.data;
                            
                            // 1. 保存到 sessionStorage
                            sessionStorage.setItem("logInfoMB", encrypt(JSON.stringify(res.data)));
                            
                            // 2. 跳转到首页
                            this.$router.push('/home');
                        } else {
                            // 登录失败
                            this.$message.error(res.message || '登录失败');
                        }
                    }).finally(() => {
                        this.loginLoading = false;
                    });
                }
            });
        },
        
        // 2. 用户名失焦后获取手机号（用于发送验证码）
        getUserMobile() {
            const username = this.form.getFieldValue('username');
            if (username) {
                api.getUserMobile({ username }).then(res => {
                    if (res.status === 0) {
                        this.mobile = res.data.mobile;
                    }
                });
            }
        },
        
        // 3. 发送验证码
        sendCode() {
            if (!this.mobile) {
                this.$message.warning('请先输入用户名');
                return;
            }
            
            this.codeLoading = true;
            this.codeButtonText = '60s';
            
            api.sendUserVerifCodeMb({ mobile: this.mobile }).then(res => {
                if (res.status === 0) {
                    this.$message.success('验证码已发送');
                    // 开始倒计时
                    this.startCountdown();
                }
            }).finally(() => {
                this.codeLoading = false;
            });
        },
        
        // 4. 倒计时逻辑
        startCountdown() {
            let seconds = 60;
            const timer = setInterval(() => {
                seconds--;
                this.codeButtonText = `${seconds}s`;
                if (seconds <= 0) {
                    clearInterval(timer);
                    this.codeButtonText = "发送验证码";
                }
            }, 1000);
        },
        
        // 5. 打开重置密码弹窗
        openReset() {
            this.resetModal = true;
            this.getCaptchaImgMB('reset');  // 获取图形验证码
        },
        
        // 6. 重置密码
        handleReset() {
            api.resetPassword({
                username: this.resetForm.getFieldValue('reUsername'),
                captcha: this.resetForm.getFieldValue('captcha')
            }).then(res => {
                if (res.status === 0) {
                    this.$message.success('重置密码邮件已发送，请查收');
                    this.resetModal = false;
                }
            });
        }
    }
}
```

## 登录流程图

```
┌─────────────────────────────────────────────────────────────────┐
│                        用户登录流程                              │
└─────────────────────────────────────────────────────────────────┘

    用户输入账号密码
           ↓
    点击登录按钮
           ↓
    ┌─────────────────┐
    │ 调用登录接口    │
    │ api.login()    │
    └────────┬────────┘
             ↓
    ┌─────────────────┐
    │ 后端验证身份    │
    │ 返回 token      │
    └────────┬────────┘
             ↓
    ┌─────────────────┐     ┌─────────────────┐
    │ 登录成功        │ ──→ │ 登录失败        │
    └────────┬────────┘     └─────────────────┘
             ↓                       ↓
    保存用户信息到 sessionStorage    显示错误信息
             ↓
    跳转到 /home 首页
             ↓
    App.vue 检测到已登录
             ↓
    调用 getRoleMenusAsync 加载菜单
             ↓
    显示系统主界面
```

## 知识点

### 🔹 v-decorator 表单绑定
```html
<a-input v-decorator="['username', { rules: [...] }]" />
```
- Ant Design Vue 的表单绑定方式
- `validateFields()` 验证并获取表单值

### 🔹 图形验证码流程
```
用户点击验证码图片 → 刷新图形验证码 → 用户识别输入 → 提交验证
```

### 🔹 Token 的作用
```
Token = 登录后的"通行证"

每次请求带上 token → 后端验证 token → 返回对应用户的数据
```

### 🔹 加密存储
```javascript
sessionStorage.setItem("logInfoMB", encrypt(JSON.stringify(res.data)));
```
- 用户信息加密后存储
- 防止信息泄露
