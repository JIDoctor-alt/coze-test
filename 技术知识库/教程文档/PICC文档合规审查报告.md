| 文档 | 整改方向 |

### 5.2 表述转换对照表

| 违规表述 | 替换为 |
|----------|--------|
| **优化方案** | 问题分析 / 设计原理 |
| **优化建议** | 风险提示 / 性能考量 |
| **重构建议** | 代码设计分析 |
| **修复前/后代码** | 问题代码解析 |
| **建议改为** | 理解其工作原理 |
| **应改为** | 这种实现方式是... |
| **可直接使用** | 理解其实现方式 |
| **实施步骤** | 问题理解要点 |
| **修复方案** | 问题原因分析 |

### 5.3 安全修复类文档整改模板

**原结构**：
```markdown
## 问题描述
[问题说明]

### 修复方案
[步骤化修复指南]

### 修复代码
[提供修复代码]
```

```markdown
## 问题描述
[问题说明]

### 问题原理分析
[该问题的技术原理是什么]

### 风险等级
[风险说明]

### 相关代码理解
[现有代码的工作原理，而非修复代码]
```
