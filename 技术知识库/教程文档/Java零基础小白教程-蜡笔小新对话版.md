#### 💻 代码示例

```java
public class DataTypesDemo {
    public static void main(String[] args) {
        // 整数类型
        byte smallNumber = 100;        // 小数字
        int normalNumber = 1000000;     // 普通整数（最常用）
        long bigNumber = 999999999L;  // 大数字，记得加L
        
        // 浮点类型（小数）
        float price = 19.99f;           // 单精度，要加f
        double score = 99.5;            // 双精度（更精确）
        
        // 字符类型
        char grade = 'A';               // 单个字符，用单引号
        
        // 布尔类型
        boolean isHuman = true;         // 要么true，要么false
        
        // 字符串（注意：String不是基本类型，是引用类型）

}
```

#### 💻 代码示例

```java
public class ControlFlowDemo {
    public static void main(String[] args) {
        
        // if-else 选择结构
        int age = 5;
        
        if (age < 6) {
            System.out.println("上幼儿园");
        } else if (age < 12) {
            System.out.println("上小学");
        } else {
            System.out.println("更大");
        }
        
        // for 循环
        for (int i = 1; i <= 5; i++) {
            System.out.println("第" + i + "次：加油！");
        }
        
        // while 循环
        int guess = 0;
        while (guess != 7) {
            guess++;
        }
        System.out.println("猜对了！答案是" + guess);
    }
}
```

---

#### ⚠️ 小白易懵点

**广志**（敲黑板）：

1. **if后面的括号里必须是boolean值**：`if(x = 5)`是错误的，应该是`if(x == 5)`！

2. **switch只能判断byte、short、int、char、String**！不能判断double、boolean！

3. **每个case后面要加break**，否则会"穿透"到下一个case！

4. **for循环的三个部分都可以省略**，但分号不能省：`for(;;)`表示无限循环！

**小新**（举手）：什么是穿透？

#### 💻 代码示例

```java
public class ArrayDemo {
    public static void main(String[] args) {
        
        // 创建数组
        int[] scores = new int[5];  // 创建一个能存5个整数的数组
        scores[0] = 90;            // 给第一个位置赋值
        scores[1] = 85;            // 给第二个位置赋值
        
        // 直接赋值
        String[] fruits = {"苹果", "香蕉", "橙子", "葡萄"};
        
        // 数组长度
        System.out.println("水果数量：" + fruits.length);  // 4
        
        // 遍历数组
        for (int i = 0; i < fruits.length; i++) {
            System.out.println("第" + i + "个水果：" + fruits[i]);
        }
        
        // 增强for循环
        for (String fruit : fruits) {
            System.out.println("水果：" + fruit);
        }
    }
}
```

---

#### ⚠️ 小白易懵点

**广志**（敲黑板）：

1. **数组下标从0开始！** 第一个元素是`arr[0]`，不是`arr[1]`！

2. **数组长度固定不变！** 创建后不能增加或减少长度！

3. **数组下标不能越界！** 如果数组长度是5，访问`arr[5]`会报`ArrayIndexOutOfBoundsException`！

4. **数组是引用类型！** `int[] arr`声明的是一个引用，指向堆内存中的实际数组！

**小新**（举手）：爸爸，越界是什么感觉？

**广志**：就像你坐火车，明明只有5节车厢，你非要去第6节车厢找座位，结果摔到铁轨上了！

**小新**（惊恐）：好可怕！我不去第6节车厢！

### 第十五课：集合框架（Java的超市货架）

**【场景】超市，小新推着购物车**

**小新**（看着货架）：哇！好多东西！

**广志**（指着不同货架）：

| 货架类型 | Java集合 | 特点 |
|----------|----------|------|
| 排队结账 | **List** | 有顺序，有编号，可重复 |
| 购物清单 | **Set** | 去重，不关心顺序 |
| 储物柜 | **Map** | 键值对，每个柜子有编号 |

**妮妮**（指着List货架）：苹果、香蕉、苹果...List可以有重复！

**正男**（指着Set货架）：清单上苹果只出现一次！

#### 💻 代码示例

```java
import java.util.*;

public class CollectionDemo {
    public static void main(String[] args) {
        
        // List：有序、可重复
        List<String> fruits = new ArrayList<>();
        fruits.add("苹果");
        fruits.add("香蕉");
        fruits.add("苹果");  // 可以重复
        System.out.println("水果列表：" + fruits);
        
        // Set：无序、去重
        Set<String> animalSet = new HashSet<>();
        animalSet.add("狗");
        animalSet.add("猫");
        animalSet.add("狗");  // 重复的不会被添加
        System.out.println("动物集合：" + animalSet);
        
        // Map：键值对
        Map<String, Integer> scoreMap = new HashMap<>();
        scoreMap.put("语文", 90);
        scoreMap.put("数学", 85);
        System.out.println("语文成绩：" + scoreMap.get("语文"));
    }
}
```

---

#### ⚠️ 小白易懵点

**广志**（敲黑板）：

1. **List、Set、Map都是接口**，不能直接实例化，需要用实现类！

2. **HashSet保证不了顺序**，如果需要顺序用LinkedHashSet或TreeSet！

3. **Map不是Collection的子接口**，它是独立的！

4. **集合只能存对象**，基本类型会自动装箱（int → Integer）！

**小新**（举手）：什么是ArrayList和HashSet？

### 第十七课：HashMap原理（图书馆找书）

**广志**（指着书架）：找到对应书架后，如果书太多（哈希冲突），就用链表或红黑树排列！

#### ⚠️ 小白易懵点

**广志**（敲黑板）：

#### ⚠️ 小白易懵点

2. **RuntimeException不需要throws声明**，也不需要try-catch！

3. **finally块一定会执行**，即使try或catch里有return！

4. **throw和throws的区别**：throw是抛出异常对象，throws是声明方法可能抛出的异常！
