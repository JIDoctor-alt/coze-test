## 🎯 知识点对应表

### K8s资源对象比喻

| K8s概念 | 蜡笔小新比喻 | 关键词 |
|---------|--------------|--------|
| Pod | 胶囊旅馆房间 | 最小单位、共享网络存储 |
| Deployment | 工厂流水线 | 滚动更新、回滚、扩缩容 |
| StatefulSet | 银行柜员 | 稳定标识、独立存储 |
| DaemonSet | 物业保安 | 每节点一个 |
| Job | 外卖订单 | 一次性任务 |
| CronJob | 定时闹钟 | 定时执行 |

### K8s网络比喻

| K8s概念 | 蜡笔小新比喻 | 关键词 |
|---------|--------------|--------|
| Service | 电话总机 | 统一入口、负载均衡 |
| Ingress | 大楼大堂 | HTTP/HTTPS路由 |
| NetworkPolicy | 门禁系统 | 入口出口规则 |
| CoreDNS | 电话簿 | 服务名→IP |

### K8s配置存储比喻

| K8s概念 | 蜡笔小新比喻 | 关键词 |
|---------|--------------|--------|
| ConfigMap | 公告栏 | 非敏感配置 |
| Secret | 保险柜 | 敏感信息、Base64编码 |
| PV | 停车位 | 持久存储 |
| PVC | 停车证 | 存储申请 |
| Volume | 收纳箱 | 临时存储 |
