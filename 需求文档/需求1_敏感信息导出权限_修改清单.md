### 三、picc-mzmtb-server 仓库修改

#### 3.1 MtbDeclareListYLServiceImpl.java

**文件路径：** `mtb-yh/mtb-yl/src/main/java/com/picchealth/module/mtb/service/impl/MtbDeclareListYLServiceImpl.java`

**修改内容：** 重写导出方法，添加权限检查和敏感信息加密

```java
/**
 * 重写父类导出方法，新增敏感信息导出隐藏权限控制
 * 
 * 业务规则：
 * 1. 根据当前操作人账号判断是否为配置了导出权限的账号
 * 2. 配置了导出权限的账号，导出信息不做加密处理
 * 3. 未配置导出权限的账号，导出信息中的敏感字段做加密处理
 */
@Override
public ApiResponse<ExportMbdeclareInfoDto> exportMbdeclareInfo(QueryMbDeclareListVo queryMbDeclareListVo) {
    // 1. 获取当前用户账号
    String userAccount = UserUtils.getUser().getUserAccount();
    
    // 2. 检查当前用户是否有榆林地区导出权限
    boolean hasExportPermission = checkYlExportPermission(userAccount);
    log.info("榆林申报综合查询导出 - 用户账号: {}, 是否有导出权限: {}", userAccount, hasExportPermission);
    
    // 3. 调用父类方法获取查询数据
    ApiResponse<ResultPage<VipMbdeclareInfoDto>> page = this.queryVipMbdeclareInfoList(queryMbDeclareListVo);
    List<VipMbdeclareInfoDto> vipMbdeclareInfoList = page.getData().getData();
    
    // 4. 如果没有导出权限，对敏感字段进行加密处理
    if (!hasExportPermission) {
        log.info("用户无导出权限，开始对敏感字段进行加密处理");
        handleSensitiveInfoEncryption(vipMbdeclareInfoList);
    } else {
        log.info("用户有导出权限，导出原始数据");
    }
    
    // 5. 生成Excel文件
    List<String> idList = queryMbDeclareListVo.getIdList();
    ExportMbdeclareInfoDto exportMbdeclareInfoDto = new ExportMbdeclareInfoDto();
    
    UnitConfig unitConfig = UnitConfigEnum.fromFlag(FlagUtils.getFlagLocal().getFlag());
    ExcelService excelService = SpringContext.getApplicationContext().getBean(unitConfig.getService2(), ExcelService.class);
    
    if (idList == null || idList.isEmpty()) {
        exportMbdeclareInfoDto = excelService.exportMbdeclareInfo(vipMbdeclareInfoList);
    } else {
        exportMbdeclareInfoDto = excelService.exportMbdeclareInfoWithId(vipMbdeclareInfoList, idList);
    }
    
    return ApiResponse.ok(exportMbdeclareInfoDto);
}

/**
 * 检查榆林地区导出权限
 * 通过查询 PersonDivision 表中当前用户的 isexport 字段判断是否有导出权限
 */
private boolean checkYlExportPermission(String userAccount) {
    try {
        PersonDivision personDivision = new PersonDivision();
        personDivision.setAccount(userAccount);
        personDivision.setFlag(UnitConfigEnum.YL.getFlag());
        personDivision.setDr(DrEnum.VALID.getValue());
        personDivision.setIsexport("1");
        
        List<PersonDivision> personDivisions = tPersonDivisionService.selectAll(personDivision);
        return personDivisions != null && !personDivisions.isEmpty();
    } catch (Exception e) {
        log.error("检查榆林导出权限失败，用户账号: {}", userAccount, e);
        return false;
    }
}

/**
 * 处理敏感信息加密
 */
private void handleSensitiveInfoEncryption(List<VipMbdeclareInfoDto> vipMbdeclareInfoList) {
    if (vipMbdeclareInfoList == null || vipMbdeclareInfoList.isEmpty()) {
        return;
    }
    
    for (VipMbdeclareInfoDto dto : vipMbdeclareInfoList) {
        // 加密参保人姓名：张三 -> 张**
        if (StringUtils.isNotBlank(dto.getName())) {
            dto.setName(encryptName(dto.getName()));
        }
        
        // 加密手机号码：15409287181 -> 154****2617
        if (StringUtils.isNotBlank(dto.getMobile())) {
            dto.setMobile(encryptMobile(dto.getMobile()));
        }
        
        // 加密身份证号：130180199802180918 -> 130182********0918
        if (StringUtils.isNotBlank(dto.getIdcard())) {
            dto.setIdcard(encryptIdCard(dto.getIdcard()));
        }
        
        // 加密专家姓名
        if (StringUtils.isNotBlank(dto.getSecondusername())) {
            dto.setSecondusername(encryptName(dto.getSecondusername()));
        }
        
        // 加密专家手机号
        if (StringUtils.isNotBlank(dto.getSecondmobile())) {
            dto.setSecondmobile(encryptMobile(dto.getSecondmobile()));
        }
    }
}

/**
 * 加密姓名：张三 -> 张**
 */
private String encryptName(String name) {
    if (StringUtils.isBlank(name) || name.length() < 1) {
        return name;
    }
    return name.substring(0, 1) + "**";
}

/**
 * 加密身份证号：130180199802180918 -> 130182********0918
 */
private String encryptIdCard(String idCard) {
    if (StringUtils.isBlank(idCard) || idCard.length() < 10) {
        return idCard;
    }
    return idCard.substring(0, 6) + "********" + idCard.substring(idCard.length() - 4);
}

/**
 * 加密手机号码：15409287181 -> 154****2617
 */
private String encryptMobile(String mobile) {
    if (StringUtils.isBlank(mobile) || mobile.length() < 7) {
        return mobile;
    }
    return mobile.substring(0, 3) + "****" + mobile.substring(mobile.length() - 4);
}
```

## 加密规则

| 字段 | 原始值 | 加密后 | 规则说明 |
|------|--------|--------|---------|
| 参保人姓名 | 张三 | 张** | 保留姓氏 + ** |
| 手机号码 | 15409287181 | 154****2617 | 前3位 + **** + 后4位 |
| 身份证号 | 130180199802180918 | 130182********0918 | 前6位 + ******** + 后4位 |
| 专家姓名 | 李四 | 李** | 保留姓氏 + ** |
| 专家手机号 | 13912345678 | 139****5678 | 前3位 + **** + 后4位 |
