## 服务调用方式详解

### 方式一：common-call 统一服务调用（推荐）

项目已有专门的 **common-call** 模块，通过数据库配置管理所有外部服务调用。

#### 1. 核心组件

```java
// 1. BaseCallService - 服务调用接口
public interface BaseCallService {
    LinkRuturnEntity callService(String serviceCode, JSON param);
}

// 2. RestCallUtils - REST 调用工具
@Component
public class RestCallUtils {
    @Autowired
    private RestTemplate restTemplate;
    
    public <T> ResponseEntity<T> exchange(
        HttpHeaders headers, 
        LinkCallConfig callConfig,  // 数据库配置
        JSONObject requestParam,
        Class<T> responseClass
    ) {
        return restTemplate.exchange(
            callConfig.getCallurl(),           // URL 从配置读取
            HttpMethod.valueOf(callConfig.getMenthod()),  // 方法从配置读取
            formEntity, 
            responseClass
        );
    }
}
```

#### 2. 数据库配置表 (link_call_config)

| 字段 | 说明 | 示例 |
|------|------|------|
| `code` | 服务编码 | `GET_USER_INFO` |
| `callurl` | 调用地址 | `http://localhost:9093/Login/getUser` |
| `menthod` | HTTP方法 | `POST` / `GET` |
| `requesttype` | 请求类型 | `REST` / `AXIS` |
| `state` | 状态 | `1`=启用, `2`=模拟数据 |
| `requestparam` | 请求参数模板 | `{"userId":""}` |
| `responsestr` | 模拟返回数据 | 测试用 |

#### 3. 使用示例

```java
@Service
public class UserService {
    
    @Autowired
    private BaseCallService baseCallService;  // 注入服务调用
    
    /**
     * 调用其他服务获取用户信息
     */
    public UserInfo getUserInfo(String userId) {
        // 1. 构造请求参数
        JSONObject param = new JSONObject();
        param.put("userId", userId);
        
        // 2. 调用服务（通过数据库配置的 code）
        //    会自动从 link_call_config 表查询 code=GET_USER_INFO 的配置
        //    然后调用配置的 URL
        LinkRuturnEntity result = baseCallService.callService("GET_USER_INFO", param);
        
        // 3. 处理结果
        if (result.isSuccess()) {
            return result.getData().toJavaObject(UserInfo.class);
        } else {
            throw new RuntimeException(result.getMessage());
        }
    }
}
```

#### 4. 配置数据库记录

```sql

INSERT INTO link_call_config (
    code,           -- 服务编码
    callurl,        -- 调用地址
```

### 方式二：HttpForwardUtil 转发（Gateway/Auth 使用）

```java
@Component
public class HttpForwardUtil {
    @Value("${forwardUrl:http://localhost:9091}")
    private String prefix;
    
    @Autowired
    private RestTemplate restTemplate;

    public <T> T post(String url, HttpServletRequest request, Object body, Class<T> clazz) {
        return restTemplate.exchange(
            prefix + url,  // 完整URL = 配置地址 + 接口路径
            HttpMethod.POST, 
            new HttpEntity(body, headers), 
            clazz
        ).getBody();
    }
}
```

### 方式三：直接使用 RestTemplate

```java
@Service
public class DirectCallService {
    
    @Autowired
    private RestTemplate restTemplate;
    
    public ApiResponse callAuth(String userId) {
        String url = "http://localhost:9093/Login/getUser";
        
        Map<String, String> body = new HashMap<>();
        body.put("userId", userId);
        
        return restTemplate.postForEntity(url, body, ApiResponse.class).getBody();
    }
}
```

## 如何在项目中使用服务调用

### 场景：Server 需要调用 Auth 的登录接口

#### 步骤1：确保依赖 common-call

```xml
<!-- server/pom.xml -->
<dependency>
    <groupId>com.picchealth</groupId>
    <artifactId>common-call</artifactId>
    <version>${project.version}</version>
</dependency>
```

#### 步骤2：在数据库配置服务

```sql
INSERT INTO link_call_config (
    code, callurl, menthod, requesttype, state, name
) VALUES (
    'AUTH_LOGIN',                              -- 服务编码
    'http://localhost:9093/Login/doLogin',     -- Auth服务地址
    'POST',                                     -- POST请求
    'REST',                                     -- REST类型
    '1',                                        -- 启用
    '认证服务-登录'
);
```

#### 步骤3：在代码中调用

```java
@Service
public class MyService {
    
    @Autowired
    private BaseCallService baseCallService;
    
    public void login(String username, String password) {
        JSONObject param = new JSONObject();
        param.put("username", username);
        param.put("password", password);
        
        // 通过服务编码调用
        LinkRuturnEntity result = baseCallService.callService("AUTH_LOGIN", param);
        
        if (result.isSuccess()) {
            System.out.println("登录成功: " + result.getData());
        } else {
            System.out.println("登录失败: " + result.getMessage());
        }
    }
}
```

## 总结对比

| 调用方式 | 使用场景 | 特点 |
|---------|---------|------|
| **common-call** | Server 调用外部服务 | 数据库配置管理，支持模拟测试 |
| **HttpForwardUtil** | Gateway/Auth 转发 | 转发原始请求，保留Header |
| **RestTemplate** | 简单直接调用 | 无额外配置，代码直接写死URL |

## 推荐使用方式

1. **Server 调用 Auth/其他服务**：使用 **common-call**（统一配置管理）
2. **Gateway/Auth 转发到 Server**：使用 **HttpForwardUtil**（保留原始请求信息）
3. **简单测试场景**：使用 **RestTemplate**（快速验证）
