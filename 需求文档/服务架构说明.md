## 数据共享方式

三个服务**直接访问同一个数据库**，不通过 HTTP 调用：

| 表 | server | user | auth | 说明 |
|----|:------:|:----:|:----:|------|
| `person_division` | ✅ | ✅ | ❌ | 权限归属表 |
| `privilege_user_info` | ✅ | ✅ | ✅ | 用户信息表 |
| `privilege_role_info` | ✅ | ✅ | ✅ | 角色表 |

## 代码示例

### 方式一：直接数据库访问（推荐，当前项目使用）

```java
// server 服务中直接查询 person_division 表
@Service("personDivisionService")
public class PersonDivisionServiceImpl extends BaseServiceImpl<PersonDivision> {
    
    public PersonDivision getByAccount(String account) {
        PersonDivision query = new PersonDivision();
        query.setAccount(account);
        query.setFlag("15"); // 榆林
        return this.selectOne(query);  // 直接查数据库
    }
}
```

### 方式二：如果需要 HTTP 调用（不推荐，当前项目未使用）

如果未来需要通过 HTTP 调用，可以使用以下方式：

#### 1. RestTemplate 方式

```java
// server 服务调用 user 服务
@Service
public class UserRemoteService {
    
    @Autowired
    private RestTemplate restTemplate;
    
    @Value("${user.service.url:http://picc-mzmtb-user:8080}")
    private String userServiceUrl;
    
    public PersonDivision getPersonDivision(String account) {
        String url = userServiceUrl + "/api/personDivision/getByAccount?account=" + account;
        ResponseEntity<PersonDivision> response = restTemplate.getForEntity(url, PersonDivision.class);
        return response.getBody();
    }
}
```

#### 2. Feign 方式（需要添加依赖）

```xml
<!-- pom.xml -->
<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-starter-openfeign</artifactId>
</dependency>
```

```java
// 启动类
@EnableFeignClients
@SpringBootApplication
public class ServerApplication {
    public static void main(String[] args) {
        SpringApplication.run(ServerApplication.class, args);
    }
}

// Feign Client
@FeignClient(name = "picc-mzmtb-user", url = "${user.service.url}")
public interface UserClient {
    
    @GetMapping("/api/personDivision/getByAccount")
    PersonDivision getByAccount(@RequestParam("account") String account);
}

// 使用
@Service
public class SomeService {
    @Autowired
    private UserClient userClient;
    
    public void doSomething(String account) {
        PersonDivision pd = userClient.getByAccount(account);
        // 使用 pd 对象
    }
}
```

## 注意事项

1. **数据一致性**：三个服务共享数据库，需要确保数据结构同步
2. **事务边界**：每个服务独立管理事务，跨服务事务需要考虑分布式事务
3. **实体类同步**：三个服务的实体类需要保持字段一致
   - 本次需求需要在 user 和 server 两个仓库的 `PersonDivision.java` 中都添加 `isexport` 字段
